# CSVычислитель

Консольная утилита на языке C для чтения, вычисления и отображения CSV-файлов, содержащих арифметические выражения.

## Особенности
- Парсинг CSV-файлов с произвольным количеством строк и столбцов.
- Поддержка целых чисел и выражений формата `= ARG1 OP ARG2`.
- `ARG1` и `ARG2` могут быть целыми числами или ссылками на ячейки (например, `A1`, `Cell30`).
- Поддерживаемые операции: `+`, `-`, `*`, `/`.
- Оптимизирован для больших таблиц: использование бинарного поиска (O (log N)) для поиска ссылок.
- Надежное обнаружение ошибок:
    - Деление на ноль.
    - Циклические зависимости ячеек.
    - Некорректные ссылки на ячейки.
    - Ошибки синтаксического анализа.
- Использует только стандартную библиотеку C.

## Требования
- CMake (версия 3.10 или выше)
- Компилятор C (gcc, clang или msvc)

## Инструкции по сборке

### Linux / macOS
```bash
cmake .
make
```

### Windows (cmd.exe)
```cmd
cmake .
cmake --build . --config Release
```

## Использование
```bash
./csvreader <имя_файла.csv>
```

### Пример
```bash
./csvreader example.csv
```

Входные данные (`example.csv`):
```csv
,A,B,Cell
1,1,0,1
2,2,=A1+Cell30,0
30,0,=B1+A1,5
```

Выходные данные:
```csv
,A,B,Cell
1,1,0,1
2,2,6,0
30,0,1,5
```

## Тестирование
Тестовые примеры находятся в директории `tests/`:
- `tests/test_errors.csv`: деление на ноль, некорректные ссылки.
- `tests/test_neg.csv`: выражения с отрицательными числами.
- `tests/test_circular.csv`: циклические зависимости.
